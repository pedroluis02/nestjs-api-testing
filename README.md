# NestJS API Example

NestJS API project structure with Clean Architecture Implementation.

## Topics

- Restful API
- Clean Architecture
- Testing

## Dependency installation

```bash
$ yarn install
```

## Running the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
